## 📖 Overview
This repository contains the resources and code developed during the **Google Developer Groups (GDG) Build with AI Workshop** at **Universidad del Norte, Barranquilla**. The focus of the session was to design and deploy an **AI-powered agent** capable of responding intelligently in real time, using modern cloud and development tools.

## 🎯 Learning Goals
By following the materials in this repository, you will learn how to:
- Create an AI agent from scratch.
- Deploy the agent to run in production environments.
- Connect models with real-world data sources.
- Interact with the agent and extract value from practical use cases.

## 🛠️ What You Will Build
A functional AI agent that:
- Processes information using AI models.
- Responds to user queries with contextual intelligence.
- Demonstrates integration between backend services and cloud infrastructure.

## 💡 Benefits
- Move from theory to practice in under 2 hours.
- Understand how applied AI works in real-world products.
- Acquire a highly demanded skill: **AI + ADK (AI Development Kit)**.
- Leave with a working project to showcase in your portfolio.
- Gain insights into building your own AI-driven solutions or startups.

## 👤 Target Audience
This workshop is ideal for:
- Beginners in technology curious about AI.
- Developers looking to integrate AI into projects.
- Entrepreneurs aiming to build intelligent products.
- Enthusiasts of **Google Cloud Platform (GCP)**, cloud computing, or automation.

## 📂 Repository Structure
- `main.py` — Entry point for the AI agent.
- `docker-compose.yml` — Containerized environment setup.
- `.env.example` — Environment variable template.
- `STEPS.md` — Step-by-step guide for reproducing the workshop exercises.
- `README.md` — Documentation and event context.

## 🚀 Getting Started

Clone the repository and set up the environment:
```bash
git clone https://github.com/jdomingu19/gdg-uninorte-build-ai-agents.git
cd gdg-uninorte-build-ai-agents
```

Install dependencies and run the agent:
```bash
pip install -r requirements.txt
python main.py
```

Or use Docker:
```bash
docker compose up
```
